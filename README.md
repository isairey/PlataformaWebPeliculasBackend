# 🌌 Acerca del proyecto

**React Movie** es una aplicación web responsive creada con React.js que consume información desde una API de películas para mostrar contenido dinámico y moderno.

La plataforma permite:

- 🎬 Explorar películas
- 🔍 Buscar contenido
- ⭐ Ver puntuaciones
- 🖼️ Mostrar posters dinámicos
- 📱 Navegación responsive
- ⚡ Interfaz rápida
- 🌐 Integración con APIs
- 🎞️ Experiencia cinematográfica moderna

# ⚡ Instalación

## 📋 Requisitos

- Node.js
- NPM
- VS Code
- Navegador moderno

---

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaWebPeliculasBackend.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaWebPeliculasBackend
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar proyecto

```bash
npm run dev
```

---

## 5️⃣ Abrir navegador

```bash
http://localhost:3000
```
